Analítica

Análisis de KPIs de campañas

Monitorizo y analizo los indicadores clave de las campañas para optimizar presupuesto, segmentación y creatividades. Trabajo con datos reales de plataformas como Facebook Ads, Instagram Ads, Google Ads y dashboards en Excel / Power BI.

Costo por clic CPC

Mide cuánto pagas por cada clic. Lo uso para comparar anuncios, audiencias y ubicaciones, detectando dónde el presupuesto es más eficiente.

  • Comparación entre campañas y conjuntos de anuncios.
  • Identificación de creativos caros o poco relevantes.
Costo por lead CPL

Indica cuánto cuesta conseguir cada registro o contacto. Es clave para cursos, servicios y generación de prospectos.

  • Relación directa con el precio del producto/curso.
  • Permite ajustar ofertas, segmentación y landings.
Tasa de clics CTR

Porcentaje de personas que hacen clic sobre las impresiones. Me ayuda a evaluar si el mensaje y el diseño están llamando la atención.

  • Si el CTR es bajo, pruebo nuevos copies y creativos.
  • Permite detectar fatiga de anuncio y momentos para renovar campañas.
Retorno de inversión publicitaria ROAS

Relación entre lo invertido y lo generado. Permite saber si la campaña es rentable y qué canales aportan más valor.

  • Seguimiento por campaña, conjunto y anuncio.
  • Apoyo decisiones de escalar o pausar campañas.

Cómo trabajo los datos

  • Descarga e integración de datos desde Facebook Ads / Google Ads.
  • Tablas dinámicas en Excel / Google Sheets para ver rendimiento por canal.
  • Dashboards en Power BI para visualizar CPL, ROAS y tendencias por periodo.
  • Recomendaciones accionables: subir/bajar presupuesto, cambiar audiencias, test A/B de creativos.

Análisis de datos

Además de configurar y optimizar campañas, trabajo el análisis de datos para entender qué funciona, qué no y dónde es mejor invertir el presupuesto. Uso Excel, Google Sheets, Power BI y Python para transformar datos en decisiones claras.

  • Construcción de reportes de rendimiento por canal, campaña y creatividad.
  • Uso de tablas dinámicas para analizar CPL, CTR, ROAS y tendencias por periodo.
  • Detección de campañas poco rentables y reasignación de presupuesto.
  • Segmentación de datos por público, dispositivo, ubicación y formato.

Mi objetivo es que los datos no se queden en un archivo, sino que sirvan para tomar decisiones concretas: qué pausar, qué escalar y qué probar.

Flujo típico de análisis

  • 1. Exportar datos desde Meta Ads / Google Ads.
  • 2. Limpiar y organizar la información en Excel / Sheets.
  • 3. Construir vistas por objetivos (leads, ventas, awareness).
  • 4. Visualizar en dashboards (Power BI / tablas dinámicas).
  • 5. Generar recomendaciones accionables para el siguiente ciclo.

Ejemplos de insights

  • Identificar horarios y días con mejor rendimiento.
  • Detectar creativos con alto CTR pero CPL elevado.
  • Comparar desempeño entre tipos de públicos (lookalike, intereses, remarketing).
  • Medir impacto de cambios en landings y formularios.

Ejemplos de gráficas generadas con Python

Analizador activo

Estas gráficas son ejemplos de análisis realizados con Python (pandas y matplotlib), exportados a imagen para usarlos en reportes y dashboards.

Procesando datos ...

Ver ejemplo de código en Python 

import pandas as pd
import matplotlib.pyplot as plt

# Datos de ejemplo
data = {
    "mes": ["Ene", "Feb", "Mar", "Abr", "May", "Jun"],
    "cpl_antes": [75, 72, 68, 62, 58, 55],
    "cpl_despues": [55, 52, 50, 48, 46, 45],
}

df = pd.DataFrame(data)

# Gráfica de línea CPL antes vs después
plt.figure(figsize=(6, 4))
plt.plot(df["mes"], df["cpl_antes"], marker="o", label="Antes")
plt.plot(df["mes"], df["cpl_despues"], marker="o", label="Después")
plt.title("Evolución del CPL por mes")
plt.xlabel("Mes")
plt.ylabel("CPL ($)")
plt.legend()
plt.tight_layout()
plt.savefig("assets/images/charts/cpl_evolucion.png", dpi=120)
plt.close()
